# 📊 Vrinda Store Sales Analysis Dashboard Using Excel

## 📌 Project Overview

**Vrinda Store Sales Analysis Dashboard** is an end-to-end Excel data analysis project focused on analyzing the annual sales performance of Vrinda Store for the year **2022**.

The objective of this project is to help Vrinda Store understand customer purchasing behavior, identify sales trends, and generate actionable insights to improve sales performance in **2023 and beyond**.

This project includes **data cleaning, data transformation, KPI analysis, Pivot Tables, Pivot Charts, Slicers, and an interactive dynamic dashboard** built entirely in **Microsoft Excel**.

---

## 🎯 Business Objective

Vrinda Store wanted to create an **Annual Sales Report for 2022** to:

* Understand customer purchasing behavior
* Identify top-performing regions and channels
* Analyze order trends and customer demographics
* Improve business strategies for future sales growth

## 🔧 Data Cleaning & Data Transformation

The following operations were performed to prepare the dataset:

* Applied filters to identify missing/blank values
* Removed inconsistent and double-spelled data
* Corrected data formatting issues
* Extracted **Month** from dates using the **TEXT() function**
* Created a new **Age Group** column using **Nested IF() formulas**

### Age Group Classification

| Age Range | Category |
| --------- | -------- |
| 0 - 24    | Teenager |
| 25 - 49   | Adult    |
| 50+       | Senior   |

## 📊 Sample Business Insights

After analyzing the data, the following insights were discovered:

* **Women customers are more likely to purchase compared to men (~65%)**
* **Maharashtra, Karnataka, and Uttar Pradesh** are the top 3 contributing states
* **Adult Age Group (25–49 years)** contributes the highest sales (~50%)
* **Amazon, Flipkart, and Myntra** are the highest contributing sales channels

---

## 🚀 Final Business Recommendation

To improve Vrinda Store sales performance:

**Target women customers aged 25–49 years living in Maharashtra, Karnataka, and Uttar Pradesh by offering personalized ads, discounts, offers, and coupons on platforms such as Amazon, Flipkart, and Myntra.**
